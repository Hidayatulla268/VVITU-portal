Key Features

Everything a University Needs

A complete ERP with tightly scoped roles, secure data pipelines, and AI-powered analytics — all wrapped in a glassmorphism interface.

Role-Based Access Control

Highly secure dashboard routing for Students, Faculty, HODs, DEOs, and Admin — each with tailored sidebar navigation and scoped permissions.

AI Attendance Predictor

Scikit-learn linear regression analyzes student attendance records and predicts semester-end outcomes with visual trend indicators.

Unified Notification System

Multi-scoped notices targeted to everyone, specific roles, branches, classes, or single users with real-time delivery and quick compose.

Excel & PDF Exports

Dynamically generated attendance and marks reports available for download on demand via openpyxl and reportlab.

Bulk CSV Uploads

Instantly upload spreadsheets to create thousands of student profiles and populate test marks in seconds.

Advanced Security Hardening

Rate limiting (5 attempts/min per IP), HSTS, SameSite/HttpOnly session cookies, Referrer-Policy headers, and first-time password flow.

HOD Timetable Management

Publish and manage branch timetables, assign faculty to subjects/classes, and designate counselors and class teachers.

Achievements System

Students and faculty submit academic and co-curricular achievements for HOD verification and public profile display.

DEO Data Entry Panel

Branch-scoped operators add/edit students, upload marks, and edit attendance within a strict 1-day editing window.

User Roles

Quick Start

Up & Running in 5 Steps

Clone, install, migrate, seed, and launch — the entire flow takes under 5 minutes.

01
Clone & Virtualenv

Clone the repo and create a Python virtual environment.

02
Install Dependencies

Run pip install -r requirements.txt to get all packages.

03
Apply Migrations

Run python manage.py migrate to set up the database schema.

04
Seed Sample Data

Run python manage.py seed_data to create branches, accounts, timetables and attendance records.

05
Launch Dev Server

Run python manage.py runserver and open http://127.0.0.1:8000/.

☁
One-Click Render

Push to GitHub and link to Render — the render.yaml blueprint auto-provisions PostgreSQL, runs migrations, and seeds data.

# Windows — quick start
python -m venv venv && venv\Scripts\activate
pip install -r requirements.txt

python manage.py seed_data
